# 🎹 Key Journey - 琴鍵旅程

專業鋼琴學習追蹤系統，幫助你記錄每一次練習、收集經典曲目、追蹤學習進度！

![Version](https://img.shields.io/badge/version-3.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![PWA](https://img.shields.io/badge/PWA-ready-purple)

## ✨ 功能特色

### 📚 完整曲目庫
- 100+ 首經典鋼琴曲目
- 涵蓋巴洛克到現代的所有時期
- 包含練習曲、奏鳴曲、協奏曲等多種類型
- 34 位大師作曲家作品

### 🎯 學習追蹤
- 記錄每次練習時間
- 添加練習筆記和心得
- 視覺化進度統計
- 追蹤學習里程碑

### 💾 資料管理
- 自動儲存所有記錄
- 支援備份匯出
- 支援備份還原
- 資料完全隱私（儲存在本地）

### 📊 進度分析
- 收藏曲目統計
- 練習時數追蹤
- 精通曲目計數
- 學習歷程視覺化

## 📖 使用說明

### 開始使用
1. 輸入你的名字
2. 開始探索曲目
3. 收藏喜歡的曲子
4. 記錄每次練習

### 備份資料
1. 點擊右上角設定按鈕 ⚙️
2. 選擇「匯出備份」
3. 下載 JSON 備份檔案
4. 妥善保存

### 還原資料
1. 點擊設定 ⚙️
2. 選擇「匯入備份」
3. 選擇之前的備份檔案
4. 確認還原

## 🔧 本地開發

### 快速開始
1. Clone 專案
```bash
git clone https://github.com/你的帳號/key-journey.git
cd key-journey
```

2. 開啟檔案
```bash
# 直接用瀏覽器開啟 index.html
open index.html  # macOS
start index.html  # Windows
```

### 本地伺服器（可選）
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# 然後訪問 http://localhost:8000
```
